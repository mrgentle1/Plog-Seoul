# Plog Seoul
2023 서울 열린데이터광장 공공데이터 활용 모바일 앱/웹 경진대회 수상작 <br/>
 <br/>

## 프로젝트 소개
Plog Seoul은 서울시에서 제공하는 <b>서울 두드림길 공공데이터를 활용하여 플로깅 기록을 관리해주는 앱</b>입니다. <br/>
플로그 서울을 통해 플로깅 기록을 SNS로 다른 사람들과 함께 공유하고, 경쟁할 수 있습니다.
